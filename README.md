# Back-End Coursework - Docker, Docker Compose & FastAPI

Repository for my college back-end assignments and class notes, focusing on containerization with Docker and API development with FastAPI.

## 📚 Contents

- `docker/` - Dockerfiles and Docker configurations
- `docker-compose/` - Multi-container setups (FastAPI + PostgreSQL + Redis)
- `fastapi/` - FastAPI projects and exercises
- `notes/` - Lecture notes on back-end concepts
- `assignments/` - Completed coursework

## 🛠️ Technologies

- **Python 3.11+**
- **FastAPI** - Modern web framework for building APIs
- **Docker** - Containerization
- **Docker Compose** - Multi-container orchestration
- **PostgreSQL** - Relational database
- **Redis** - Caching and session management
- **SQLAlchemy** - Async ORM for Python
- **Pydantic** - Data validation

## 🚀 Getting Started

### Prerequisites

- Docker and Docker Compose installed
- Python 3.11+ (for local development)
- Git

### Running with Docker Compose

```bash
# Clone the repository
git clone https://github.com/your-username/back-end-coursework.git
cd back-end-coursework

# Start all services
docker-compose up -d

# Stop all services
docker-compose down
